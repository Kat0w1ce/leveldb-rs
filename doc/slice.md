# Slice

底层数据结构，保存一个指向u8数组的指针和长度

## From

可以从String, &str 等等数据结构初始化

## to_string

还愿为字符串

## to_str\ to_string

还原为str或者String

## compare

比较

## start_with

是否以某个前缀开头

## remove_prefix 

 Drop the first "n" bytes from this slice.
